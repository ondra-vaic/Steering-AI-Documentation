1. Minimal Setup

This sample shows a minimal setup of an entity and it's steering system. The scene contains three entities which go forward at different speeds. The sample contains the files you would get if you followed the Minimal Setup guide.

  • Scene
  • Steering System Asset
  • Prefab

There are three copies of the SteeringEntity prefab in the scene. Each has a different value for Speed on the GoForwardComponent.

SampleSteeringSystemAsset:

  • Main Tag - SampleEntityTagComponent
  • Simple Behaviors - [GoForwardJobWrapper]
  • Prefab - SteeringEntity

Behavior Components:

  • GoForwardAuthoring

Other Components:

  • SteeringEntityAuthoring
  • DesiredVelocityAuthoring
  • Movement2DAuthoring
note

Your entities will generally need to have SteeringEntityAuthoring and DesiredVelocityComponent on them. It will usually also be useful to add movement, in this case with Movement2DAuthoring.
