4. Collisions

Entity with collision resolution against regular Colliders.

Normally in Unity CharacterController handles both movement and collision resolution. Here the collision resolution is split into a separate system - CollideAndSlideSystem. This allows the collision resolution to be used with any other movement system, in this sample the Movement2DSystem. To use it, add KinematicBodyAuthoring and set the LayerMask. The entities will react to regular Unity Colliders.

  • Scene
  • Steering System Asset
  • Prefab

There are two entities in the scene, both are trying to travel straight forward using the GoForwardJobWrapper. The CollideAndSlideSystem makes thes slide along the Colliders.
