# Entity Queries

[Entity queries](https://docs.unity3d.com/Packages/com.unity.entities@1.3/manual/systems-entityquery.html) are crucial part of the [ECS](https://docs.unity3d.com/Packages/com.unity.entities@1.3/manual/index.html) package. They find all entities matching some set of [components](https://docs.unity3d.com/Packages/com.unity.entities@1.3/manual/concepts-components.html). The base system runs an entity query which matches all entities with the same *Main Tag*. Information about these entities is then packaged as [`BaseBehaviorParams`](/docs/documentation-core/base-system/BaseBehaviorParams) and passed around to all behaviors by the [`BaseSteeringSystem`](/docs/documentation-core/base-system/BaseSteeringSystem).
