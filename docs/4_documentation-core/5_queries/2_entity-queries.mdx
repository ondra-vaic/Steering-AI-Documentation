# Entity Queries

## Overview

Entity queries are from the Unity's ECS package [link]. They find all entities matching some set of components. The base system runs an entity query which matches all entities with the same *Main Tag*. Information about these entities is then packaged as [`BaseBehaviorParams`](/docs/documentation-core/base-system/BaseBehaviorParams) and passed around to all behaviors by the [`BaseSteeringSystem`](/docs/documentation-core/base-system/BaseSteeringSystem).
