# SteeringSystemAsset

Instance of `SteeringSystemAsset` is what is loaded by the `BaseSteeringSystem` and defines it's behavior. The system is defined declaratively from queries and behaviors. To create one right click into the assets and select `Create/SteeringSystem`. Double click it to open it in the editor [link] where it can be edited. 

<img src="/img/core/SteeringSystemMain.png" alt="Description of the image"/>

## Usage

1) Select the asset in the inspector and mark it as addressable. 
2) Create a script inheriting from `BaseSteeringSystem` and return the addressable path to the asset `getAssetReferenceName()`.

<img src="/img/core/AddressablePath.png" alt="Description of the image"/>

```csharp title="SampleSteeringSystem.cs"
using SteeringAI.Core;
using Unity.Entities;

[UpdateInGroup(typeof(SteeringSystemsGroup))]
public partial class SampleSteeringSystem : BaseSteeringSystem
{
    protected override string getAssetReferenceName()
    {
        return "Assets/Sample/SampleSteeringSystemAsset.asset";
    }
}
```
