# SteeringSystemAsset

Instance of `SteeringSystemAsset` is what is loaded by the [`BaseSteeringSystem`](/docs/documentation-core/base-system/BaseSteeringSystem) and defines it's behavior. The system is defined declaratively from [queries](/docs/category/queries) and [behaviors](/docs/category/behaviors). 

<img src="/img/core/SteeringSystemMain.png" alt="Description of the image" width="550px"/>
*A `SteeringSystemAsset` being edited in the [editor](/docs/documentation-core/editor).*

## Creation

Follow these steps to create a `SteeringSystemAsset`:

1) Right click in the Assets and select `Create/SteeringSystem`. 
2) Select the new `SteeringSystemAsset`, and mark it as [`Addressable`](https://docs.unity3d.com/Packages/com.unity.addressables@2.3). 
3) Double click it to open it in the [editor](/docs/documentation-core/editor) where it can be edited.
