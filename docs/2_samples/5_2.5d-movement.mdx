# 5. 2.5D Movement

*Entity following a path along terrain smoothly.* 

This sample shows usage of [`Move25DSystem`](/docs/documentation-defaults/movement/2_25D-movement). This system allows movement on ground. Paired with [`CollideAndSlideSystem`](/docs/documentation-defaults/movement/collision-resolution) it is similar to Unity's [`CharacterController`](https://docs.unity3d.com/ScriptReference/CharacterController.html). It handles steps and gradual speeding up and down depending on whether the entity travels up- or down-hill. The [`CollideAndSlideSystem`](/docs/documentation-defaults/movement/collision-resolution) makes sure the entity does not travel through colliders. It can also be used with other *Movement Systems*, it's usage shown in the [collisions sample](/docs/samples/collisions).
