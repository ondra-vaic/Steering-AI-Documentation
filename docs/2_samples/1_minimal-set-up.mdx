# 1. Minimal Setup

<video controls width="800">
  <source src={useBaseUrl("/videos/GoForwardVideo.mp4")} type="video/mp4" />
  Your browser does not support the video tag.
</video>

This sample shows a minimal setup of an *entity* and it's *steering system*. The scene contains three entities which go forward at different speeds. The sample contains the files you would get if you followed the [Minimal Setup](/docs/getting-started/minimal-set-up) guide. 

![Minimal Setup folder](/img/MinimalSetupFolder.png)

-----

<Tabs>
  <TabItem value="third" label="Scene"> 
    There are three copies of the *SteeringEntity* prefab in the scene. Each has a different value for `Speed` on the [`GoForwardComponent`](/docs/documentation-defaults/behaviors/simple-behaviors/go-foward#goforwardcomponent).
    <img src={useBaseUrl("/img/MinimalSetupScene.png")} alt="Description of the image"/>
  </TabItem>
  <TabItem value="first" label="Steering System Asset">

    - ***Main Tag*** - `SampleEntityTagComponent`
    - ***Simple Behaviors*** - [[`GoForwardJobWrapper`](/docs/documentation-defaults/behaviors/simple-behaviors/go-foward)]
    - ***Prefab*** - *SteeringEntity*

    <img src={useBaseUrl("/img/MinimalSteeringSystem.png")} alt="Description of the image"/>
  </TabItem>
  <TabItem value="second" label="Prefab"> 
    
    **Behavior Components**:

    **Other Components**:
    - [`SteeringEntityAuthoring`](/docs/documentation-core/base-system/steering-entity)
    - [`DesiredVelocityAuthoring`](/docs/documentation-defaults/Combiner#desiredvelocitycomponent)
    - [`Movement2DAuthoring`](/docs/documentation-defaults/movement/0_2D-movement)
    :::note
      Your entities will generally need to have [`SteeringEntityAuthoring`](/docs/documentation-core/base-system/steering-entity) and [`DesiredVelocityComponent`](/docs/documentation-defaults/Combiner#desiredvelocitycomponent) on them. It will usually also be useful to add movement, in this case with [`Movement2DAuthoring`](/docs/documentation-defaults/movement/0_2D-movement).
    :::
