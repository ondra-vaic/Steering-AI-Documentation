# 5. Collisions

*Entity with collision resolution against regular `Colliders`.*

Normally in Unity `CharacteController` handles both movement and collision resolution. Here the collision resolution is split into `CollideAndSlideSystem`. This allows the collision resolution to be used with any other movement system, in this sample the `Movement2DSystem`. To use it, add `KinematicBodyAuthoring` and set the `LayerMask`. The entities will react to regular Unity `Colliders`.

*The algorithm is based on paper about numerically stable collide and slide algorithm [link paper] by [name].*
