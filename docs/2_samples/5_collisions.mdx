# 5. Collisions

<video controls width="800">

*Entity with collision resolution against regular [`Collider`](https://docs.unity3d.com/ScriptReference/Collider.html)s.*

Normally in Unity [`CharacterController`](https://docs.unity3d.com/ScriptReference/CharacterController.html) handles both movement and collision resolution. Here the collision resolution is split into a separate system - [`CollideAndSlideSystem`](/docs/documentation-defaults/movement/collision-resolution). This allows the collision resolution to be used with any other movement system, in this sample the [`Movement2DSystem`](/docs/documentation-defaults/movement/0_2D-movement). To use it, add [`KinematicBodyAuthoring`](/docs/documentation-defaults/movement/collision-resolution) and set the [`LayerMask`](https://docs.unity3d.com/ScriptReference/LayerMask.html). The entities will react to regular Unity [`Collider`](https://docs.unity3d.com/ScriptReference/Collider.html)s.
